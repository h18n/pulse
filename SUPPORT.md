# Support

We are excited you are building and visualizing with Pulse! If you need help, please follow these guidelines:

## 1. Questions and Troubleshooting
If you have a general question (e.g., "How do I hook up my app with OpenTelemetry to Pulse?"), please:
1.  Check the documentation in `docs/` and the `README.md`.
2.  Start a thread in our [GitHub Discussions](https://github.com/h18n/pulse/discussions). DO NOT open an Issue for basic config questions.

## 2. Bug Reports
If you are confident you have found a reproducible bug in Pulse, please open an Issue using the **Bug Report** template. Provide detailed steps to reproduce it.

## 3. Feature Requests
If you want to request a new feature (like a new widget for the dashboard or a new OpenTelemetry processor), open an Issue using the **Feature Request** template.

## 4. Security Vulnerabilities
Do NOT open a public issue for a security vulnerability. Please refer to our `SECURITY.md` for information on how to report it confidentially.
